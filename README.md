# Walmart-Sales-Analysis
Analyzed Walmart sales data to identify trends, perform regression analysis, and forecast future sales using time series models.
## 📊 Project Highlights

- Performed exploratory data analysis (EDA)
- Built regression model for prediction
- Evaluated model using R², MAE, MSE, RMSE
- Performed time series forecasting using ARIMA/SARIMA

## 📁 Files Included

- Walmart_Sales_Analysis.ipynb → Main notebook
- Walmart_Store_sales.csv → Dataset

## 🚀 How to Run

1. Open the notebook in Google Colab
2. Run all cells
3. View analysis and results
